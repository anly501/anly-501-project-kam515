Figure 1: Word cloud of "K-12 education" news headlines in the US

The above word cloud represents popular headlines in news organizations based in the United States. The keywords used were "K-12 education". Words that appear more frequently in the headlines are larger in size. The word cloud was created using the python package wordcloud. Additionally, a count vectorizer algorithm was used to create a word frequency table. The word frequency table is viewable on the GitHub code repository.

Notably, the words "parent", "attach", and "shortage" stand out as unexpected to me as a reader. Still, this prominence stands out given larger recent discussions around the K-12 educational system and teacher shortages. It is still important to contextualize this word cloud in the fact that it was produced by data coming from the News API--a news aggregator. Thus, the word cloud is not representative of the entire population in the United States, but can serve as a good indicator for the larger discourse going on surrounding education.
